Linux Mint 20 安装微信

2020-07-28 Linux Linux 0 Comments

最近 Linux Mint 20 发布了，手痒又折腾了一下。体验了几天，个人感觉非常好，就是国产那堆应用安装麻烦了一点。感谢 deepin 对很多国产应用做了移植，适配工作，让我们可以直接在 debian 系的系统上直接安装使用。下面以安装微信为例说明一下在 debian 系的系统上怎么安装 deepin 适配过的部分国产应用。
